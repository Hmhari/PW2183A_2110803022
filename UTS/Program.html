Program Web

Program Web adalah salah satu Mata Kuliah yang ada di prodi Sistem Informasi, yang mana digunakan untuk membuat Web di sebuah aplikasi web. Program web biasanya menggunakan Laborotarium Terpadu karena Fasilitas yang ada sangat memadai dan mendukung serta dengan adanya Komputer yang sangat bagus.

Mata Kuliah ini berbaur dengan coding, jadi jika Mahasiswa tidak mampu untuk membuat program maka Terjadilah eror. Program web juga mempelajari tata cara membuat web yang bagus dan menarik sehingga para Konsumen atau pengguna gadjet tertarik untuk membaca webnya.
